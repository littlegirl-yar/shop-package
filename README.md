# shop-package
Это реализация простого магазина (многостраничное/blade) в виде пакета под Laravel. Можно установить пакет, опубликовать роуты, миграции, сидеры, конфиг. Также здесь сделан фасад для получения текущего курса валют. Фактической задачей было научиться разрабатывать пакеты под Laravel.
```
composer require yaromir/shop-package

```
